# DentResa — Live Demo（静的ビルド）

歯科専用Web予約・リコール管理システム **DentResa** プロトタイプの動作デモ。

- Next.js アプリの静的エクスポート成果物のみを配置（ソースは非公開）。
- データはサンプル（モック）。ログイン不要・ゲストで自由に操作できます。

## エンドユーザー（患者）向け Web予約フォーム
診察券番号（診察ID）を入力して予約まで進めます。
- https://tsubasagit.github.io/dentresa-demo/reservations/new/
- デモ用ID例：00231 / 00489 / 00512
