# DentResa — Live Demo（静的ビルド）

歯科専用Web予約・リコール管理システム **DentResa** プロトタイプの動作デモ。

- Next.js アプリの**静的エクスポート成果物のみ**を配置した公開用リポジトリです（ソースは非公開）。
- データはサンプル（モック）。ログイン不要・ゲストで自由に操作できます。
- 「予約管理 → ＋新規予約」から**予約入力フォーム**も体験できます。
- 公開URL: https://tsubasagit.github.io/dentresa-demo/

LP: https://tsubasagit.github.io/dentresa-lp/
